# 🏦 Bank Management System

A simple **Bank Management System** built with **Python** that performs complete **CRUD (Create, Read, Update, Delete)** operations using a JSON file as a lightweight database.

This project demonstrates the fundamentals of file handling, object-oriented programming (OOP), and data management in Python.

---

## 🚀 Features

- ✅ Create a new bank account
- 💰 Deposit money
- 💸 Withdraw money
- 👤 View account details
- ✏️ Update account information
- 🗑️ Delete an account
- 💾 Store data permanently using JSON
- 🔢 Auto-generate unique account numbers
- ✔️ CNIC validation
- ✔️ Age verification (18+)
- ✔️ Gender validation
